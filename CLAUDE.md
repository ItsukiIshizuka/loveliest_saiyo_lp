# **Project: Loveliest 求人LP制作**

千葉県我孫子市のトータルビューティーサロン「Loveliest」のスタッフ採用に特化したランディングページ（LP）制作プロジェクト。

## **1\. プロジェクト概要**

* **クライアント**: Loveliest（ラヴリエスト）  
* **場所**: 千葉県我孫子市（個人サロン）  
* **主要サービス**: ネイル・まつげ・ヘアなどトータルビューティー  
* **最終コンバージョン**: 応募・サロン見学のご連絡

## **2\. ターゲットユーザー**

* アイリスト（美容師免許保持者）  
* ペットと一緒に働きたい方  
* 自由な働き方・フレックス勤務を希望する方  
* しっかり稼ぎたい方（歩合・高単価志向）  
* 未経験・新卒・ブランクからの復帰を考えている方  
* 人間関係に悩まず長く安心して働きたい方

## **3\. サービス仕様 (USP)**

* **キャッチコピー**: 愛犬と一緒に働けるサロン / 自由な働き方で、しっかり稼ぐ  
* **お給料**:  
  * 正社員：月給 230,000円  
  * アルバイト・パート：時給 1,250円  
  * 歩合：売上60万円以上で40%還元  
* **福利厚生**:  
  * 社食無料（豆乳パン・毎日）  
  * 社会保険完備  
* **ペット同伴出勤OK**: 専用待機スペースあり（猫不可・面接時同伴必須）  
* **完全フレックス制**: 営業時間 8:30〜20:30 の中で出退勤自由  
* **未経験歓迎**: 1〜3ヶ月カリキュラム完備、営業時間内に練習可能  
* **募集職種**: アイリスト（美容師免許必須）

## **4\. コンテンツ構成案**

1. **FV (First View)**: キャッチコピー、ターゲットの共感ポイント、応募への誘導  
2. **サロンについて**: 雰囲気・客層・オーナーの強み  
3. **お給料・歩合制度**: 正社員/パート給与、歩合の仕組み  
4. **福利厚生**: 社食無料（パン）、社会保険完備  
5. **ペット同伴OK**: 専用スペース・注意事項  
6. **完全フレックス制**: 営業時間・勤務の自由度  
7. **未経験サポート**: カリキュラム・練習環境  
8. **こんな方におすすめ**: ターゲット共感リスト  
9. **応募方法・CTA**: 気軽に連絡・サロン見学歓迎

## **5\. 制作ガイドライン**

* **デザイン方針**:  
  * 清潔感、信頼感、落ち着いた大人の女性向けデザイン  
  * スマートフォン閲覧をメインとしたモバイルファースト設計  
* **技術スタック**:  
  * Next.js (App Router) + React + Tailwind CSS + TypeScript + shadcn/ui  
* **開発スキル**: landing-page-guide-v2 をメインに使用  
* **CV導線**: 応募連絡・サロン見学申込（URL/電話番号はひな型完成後に共有）
